## Declaração do CSS

### Inline stylesheet

```plaintext title="Atributo Style"
            declaration
          ┌──────┴───────┐
<p style="font-size: 20px;">Lorem ipsum dolor</p>
          └───┬───┘  └─┬─┘
          property   value
```

Atributo style="":

### Internal stylesheet


```plaintext title="Blocos de Estilo"
selector
   ┌┴┐     declaration
    p { ┌──────┴───────┐
        font-size: 20px;
    }   └───┬───┘  └─┬─┘
        property   value
```
