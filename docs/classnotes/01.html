HTML#

Tags (Sintase)#

Estrutura:

openning              closing
  tag      content      tag
  ┌┴┐┌────────────────┐┌─┴─┐
  <p>Lorem ipsum dolor.</p>
  └────────────────────────┘
          Element

Exemplos:

  • Cabeçalho: <title>, <link>, <style>, <meta>
  • Texto: <p>, <b>, <i>, <h1-6>
  • Lista: <ul>, <ol>, <li>, <dl>, <dt>, <dd>
  • Hiperlink: <a>
  • Tabela: <table>, <tr>, <th>, <td>
  • Imagem: <img>
  • Mais exemplos: HTML Reference - MDN

Atributo de tag (Sintase)#

Estrutura:

attribute (chaves="valor")
   ┌─────┴────┐
<p lang="pt-BR">Lorem ipsum dolor.</p>

Exemplos:

  • Global: lang, id, class
  • Específico:
    • <img>: src, alt
    • <ol>: start, type
    • <table>: border
    • <meta>: name, content

Código:

<meta name="description" content="A description of the page" />

Hyperlink#

Estrutura:

hyperlink = content + url


content = text, image...
<a href="url">content</a>

Exemplo:

<a href="http://www.ifpb.edu.br/">ifpb</a>

Imagem#

Estrutura:

<img src="url" alt="text alternative" />

Exemplo:

index.html => image.png

/
└── var
    └── www
        └── site
            └── pages
                ├── index.html
                └── img
                    └── image.png
